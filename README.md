# Beadwork Tool — 3-bead technique

A browser-based design tool for the **3-bead weave** (a beadwork craft from
Kutch, Gujarat). The grid is built from the real bead shape and weave geometry,
so what the designer draws on screen is what the artisan weaves — no more
squished designs from square-pixel grids.

Live: https://part-time-artist.github.io/Beadwork-3-tech/

```
npm run dev      # Vite dev server (http://localhost:3000)
npm run build    # production build → dist/
```

Docs: `BEADWORK_TOOL_SPEC.md` (geometry + feature spec) ·
`DESIGN_DECISIONS.md` (locked decisions) ·
`Beadwork Tool — Code Explained.md` (plain-language code walkthrough).

Forked long ago from [Etch](https://etch.constraint.systems/); the Etch code
itself has been fully replaced.
